Parâmetros de função

Parâmetros padrão

Se uma função for chamada com argumentos ausentes (menos que os declarados), os valores ausentes serão definidos como undefined. Às vezes isso é aceitável, mas às vezes é melhor atribuir um valor padrão ao parâmetro:

function myFunction(x, y) {
if (y === undefined) {
y = 2;
}
}

Valores de Parâmetros padrão

O ES6 permite que os parâmetros de função tenham valores padrão.

function myFunction(x, y = 10) {
return x + y;
}
myFunction(5);
